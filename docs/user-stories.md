# User Stories

- As a Product Manager, I want to see a list of features with zero usage in the last 30 days so I can stop maintaining dead code.

- As a Founder, I want to see a spike in 'plan_downgraded' events in real-time so I can reach out to those customers immediately.

- As a Growth Marketer, I want to compare DAU (Daily Active Users) against the previous week to see if our new marketing campaign worked.

- As a Software Engineer, I want a clean API endpoint and a JSON schema so I can integrate this analytics tool into my app in under 5 minutes.
